## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/Callmeg0d/Orgs_Builds_Activities
```
## Запуск через Docker

# Собрать образы
docker-compose build
# Запустить в фоновом режиме
docker-compose up -d
# Просмотр логов
docker-compose logs -f
# Остановка
docker-compose down

## Не забудьте обновить файл .env
